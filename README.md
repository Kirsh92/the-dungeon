
# The Dungeon (La Mazmorra)

**The Dungeon** is an open-source, text-based endless dungeon crawler with a dark LitRPG twist. As Subject #4.472, you must survive a brutal, randomized labyrinth while a cold, bureaucratic System and interplanar spectators judge your every move. Gather bizarre loot, uncover the dark truth left by past subjects, and try to break the cycle.

It is based on the source code of *Dungeon Crawler on Demand* by Redpangilinan, heavily modified to introduce deep lore, narrative events, and incremental survival mechanics.

The game is available for Android and Web/Desktop platforms. You can find official releases of the game on:
